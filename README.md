# analise-emiss-es-seeg
Exploratory data analysis of greenhouse gas emissions in Brazil (SEEG)

Desafio prático de análise de dados com Python e Pandas utilizando dados públicos do SEEG.
Limpeza e manipulação de dados;
Agrupamentos avançados (groupby, MultiIndex);
Identificação de máximos por estado e setor (idxmax, xs);
Interpretação e comunicação de insights.
